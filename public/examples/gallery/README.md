# Teaching gallery — shareable low-res IR frames

Eight of Noah Jefferson's infrared frames (Nikon Z50, IR-converted, red filter),
downscaled to a **1600 px long edge, JPEG q80** so users have plenty to play with
without giving away the full-resolution originals. Originals are **not** stored in
this repo — only these low-res copies ship / deploy.

These complement the three RAW lesson files (`../canopy.dng`, `../lodge.dng`,
`../hillside.dng`): the DNGs teach the true-RAW, sub-2000K white-balance path;
this gallery is the free, shareable "here, go play" set covering every color-IR
teaching surface.

## Second camera — full-spectrum Nikon D5300 (magenta look)

Four frames from a **full-spectrum-converted Nikon D5300** wearing a
**lens-mounted IR filter** (exact wavelength unknown; white-balanced in-camera),
a distinctly **magenta/purple** palette instead of the red-filter Z50 look above.
Two reasons they earn a place: a *second color world* from a different capture
(so white balance and the R⇄B swap teach differently on a full-spectrum + lens-
filter source vs. a red-filter one) and the first **non-forest subject** in the
set (the urban hillside town). Same 1600 px / q80 low-res treatment.

| File | Scene | Best for teaching |
|------|-------|-------------------|
| `magenta-woodland.jpg` | Magenta woodland, bare branches, water | Dense-scene WB, texture, reflection |
| `magenta-fir.jpg` | Dark fir among bare branches, near-black | Low-key tone, shadows, strong B&W conversion |
| `magenta-hilltown.jpg` | Urban hillside — houses, cars, cypress, flat sky | WB on man-made/neutral surfaces, sky mask on a new hue |
| `magenta-dusk-trees.jpg` | Backlit conifer + weeping tree at dusk, crimson | Foliage detail, moody tone |

Each frame carries a small **bottom-right watermark** — the NJ mark plus
`jefferson-photo-studio.pages.dev` — over a subtle scrim, so a photo shared out
of the tutorial still points back to the tool. It's deliberately corner-only
(not across the picture) and croppable by design.
