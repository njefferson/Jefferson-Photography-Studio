Add to Home Screen — one app, two, or three

Everything runs in the browser, but you can also Add to Home Screen for a full-screen, offline app with its own icon. Install this launcher, either tool on its own, or any mix — they share nothing but this page.

One app

The whole Studio

Install this launcher: one icon that opens here, then you pick Infrared or Macro.

Two apps

Each tool on its own

Install Infrared and Macro separately: two icons, each opening straight into its tool.

Three

All of the above

The launcher and both tools — three icons, if you jump between them and want the quick doors too.

On iPad / iPhone: open the page you want to install, tap Safari's Share button, then Add to Home Screen. On Android / Chrome, use the browser's Install app menu. You can add or remove any of these any time — installing one never blocks the others, and the ‹ Studio link inside each tool always brings you back here.

You're in the installed Studio app — it has no Safari Share button, and links can't leave the app either, so Add to Home Screen can't be reached from here. To install Infrared or Macro as its own app, copy its link:

Then open Safari, paste the link in the address bar, and tap Share → Add to Home Screen there.

Pick your Home-Screen icon

Choose the icon your installed Studio app will wear, then Add to Home Screen. You can change it any time: pick a new style here, remove the old tile, and add the launcher again — an installed tile keeps its icon until it's re-added.
