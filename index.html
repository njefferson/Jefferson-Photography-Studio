Install it your way — one app, two, or three

Everything runs in the browser, but you can also Add to Home Screen to get a full-screen, offline app with its own icon. Install this launcher, either tool on its own, or any mix — they share nothing but this page, so an installed Infrared editor never carries the Macro engine and vice-versa.

One app

The whole Studio

Install this launcher: one icon that opens here, then you pick Infrared or Macro. Both tools, least clutter.

Two apps

Each tool on its own

Install Infrared and Macro separately: two icons, each opening straight into its tool with no menu.

Three

All of the above

The launcher and both tools — three icons. Handy if you jump between them and also want the quick single-tap doors.

On iPad / iPhone: open the page you want to install (this launcher for the whole Studio, or open a tool below to install just that one), tap Safari's Share button, then Add to Home Screen. On Android/Chrome, use the browser's Install app menu.
Changed your mind? You can add or remove any of these any time — installing one never blocks the others, and the ‹ Studio link inside each tool always brings you back here.

Choose the icon your installed Studio app will wear, then Add to Home Screen. You can change it any time: pick a new style here, remove the old tile, and add the launcher again — an installed tile keeps its icon until it's re-added.
