Active install icon: Icon A — the page's apple-touch-icon now points at it.

2 · Add it to your Home Screen

  1. With Icon A selected, tap Safari's Share button, then Add to Home Screen. It should pre-fill the name Icon A — keep it and add.
  2. Come back here, tap Icon B, then Add to Home Screen again (name Icon B).
  3. Look at the two icons now on your Home Screen and compare.

3 · What it means

Icon A app shows the dark icon, Icon B app shows the light one → the swap works. We can build the picker straight into the install flow — pick a style, then Add to Home Screen.
Both apps show the same icon (whichever you added first) → iOS cached it for this page. Then we ship a separate little install page per style instead. Test that fallback right below.
