Macro focus stacking

A macro shot has paper-thin depth of field, so a single frame is only sharp in one slice. A focus-shift burst walks the focus through the subject; this tool measures where each frame is sharpest and blends those regions into one fully-sharp image, keeping the background bokeh smooth.

JPEG or PNG sets for now. Everything runs on your device — nothing is uploaded.

Install as an app — one, two, or three

On iPad/iPhone: open the page you want to install, tap Safari's Share button, then Add to Home Screen for a full-screen, offline app with its own icon. On Android/Chrome, use the browser's Install app menu.

You're in the installed app right now — it has no Safari Share button, and links can't leave the app either, so Add to Home Screen can't be reached from here. then open Safari, paste it in the address bar, and tap Share → Add to Home Screen there.

Macro and Infrared are separate tools under one Studio launcher (the ‹ Studio link up top goes back to it). You pick what to install:

  • This tool alone — install from here; its icon opens straight into Macro Studio.
  • The whole Studio — install from the launcher: one icon, then pick Macro or Infrared each time.
  • Both, or all three — install any mix; each is independent, so Macro never carries the Infrared editor and vice-versa.

You can add or remove any of them later — installing one never blocks the others.
